# Data table contents

## Morphometry
sp - species  
sex - male (m) or female (f)  
mat - Sexual maturity, wether the individual is juvenile (j) or adult (a)
bl - body length (cm)  
dl - disk length (cm)  
ed - distance between eyes (cm)  

## Brain measurements
tbl - total brain length (cm)  
tbw - total brain width (cm)  
tbh - total brain height (cm)  
tbwe - total brain weight (g)  
ob_l - olfatory bulb lenght (cm)  
ob_w - olfatory bulb width (cm)  
ob_h - olfatory bulb height (cm)  
ob_we - olfatory bulb weight (g)  
tel_l - telencephalon lenght (cm)  
tel_w - telencephalon width (cm)  
tel_h - telencephalon height (cm)  
tel_we - telencephalon weight (g)  
mes_l - mesencephalon lenght (cm)  
mes_w - mesencephalon width (cm)  
mes_h - mesencephalon height (cm)  
mes_we - mesencephalon weight (g)  
dien_l - diencephalon lenght (cm)  
dien_w - diencephalon width (cm)  
dien_h - diencephalon height (cm)  
dien_we - diencephalon weight (g)  
cer_l - cerebelum lenght (cm)  
cer_w - cerebelum width (cm)  
cer_h - cerebelum height (cm)  
cer_we - cerebelum weight (g)  
med_l - medulla oblongata lenght (cm)  
med_w - medulla oblongata width (cm)  
med_h - medulla oblongata height (cm)  
med_we - medulla oblongata weight (g)
