# PHD-Mateus-Alencar
PHD project by Mateus Alencar regarding sexual dimorphism in the brain organization of elasmobranchs.  
Raw data can be found in the `Data` folder`L0_data_elasmo_brain` file and a more detailed explanation of the abreviations can be found at `README_L0_data`. Processed data can be found at the `L1_data_elasmo_brain_alometry` and a more detailed explanation of the abreviations can be found at `README_L1_data`  
All scripts and plots can be found at the `Scripts` folder and their respective outputs at the `Results` folder.
